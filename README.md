## 🚀 Overview

Sistema de detección de objetos basado en YOLO para análisis de imágenes/vídeo.
Incluye evaluación de rendimiento mediante comparación con ground truth,
calculando errores de localización y área de bounding boxes.

Tecnologías: Python, OpenCV, YOLO (Ultralytics)

## 🎯 Objetivo

El sistema detecta personas y partes del cuerpo en imágenes
y evalúa la precisión del modelo comparando con anotaciones reales.
